HTML Cheat Sheat: Tables

Table Elements

Name Table Element Description
Table Row <tr> Used to add rows to a table before adding table data and table headings.
Table Data <td> can be nested inside a table row element, <tr>, to add a cell of data to a table.
Table Head <thead> defines the headings of table columns encapsulated in table rows.
Table Body <tbody> a semantic element that will contain all table data other than table heading and table footer content.
Table Heading <th> used to add titles to rows and columns of a table and must be enclosed in a table row element <td>
Table Footer <tfoor> uses table rows to give footer content or to summarize content at the end of a table.
Table <table> the table element has content that is used to represent a two-dimensional table made of rows and columns.
